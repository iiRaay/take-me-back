# Photos Folder

Add your photos here. The app will automatically:
- Read all images from this folder
- Extract GPS location data from photo metadata (EXIF)
- Display pins on the globe map for photos with location data

## Supported Formats
- JPG/JPEG
- PNG
- GIF
- WEBP
- HEIC/HEIF

## Note
Photos should contain GPS metadata (EXIF data) to appear on the map. Most smartphones and modern cameras automatically include this data when location services are enabled.
